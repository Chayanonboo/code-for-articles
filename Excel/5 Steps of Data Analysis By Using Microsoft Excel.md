## IF function

## Discount Column
```
=IF(J5>=20,"Y","N")
```

## Final Price column
```
=IF(J5>20,0.95*L5,L5)
```

## Vlookup function

## Company name column
```
=VLOOKUP(F5,'Customer Info'!$A$4:$C$12,2,FALSE)
```
-------------------------------------------------------------------
## Company name representative

=VLOOKUP(F5,'Customer Info'!$A$4:$C$12,3,FALSE)
